使用Docker镜像

什么是Docker镜像？Docer的镜像是一个只读的文件系统，除了提供容器（进程）运行时所需的程序、库、资源、配置等文件外，还包含了一些为运行时准备的一些配置参数（如匿名卷、环境变量、用户等）。镜像不包含任何动态数据，其内容在构建之后也不会改变。
